## プロジェクト概要

開発者ツール群のモノレポ。各ツールは `apps/` に配置、共通UIは `packages/ui/`。

## 構成

```
apps/                    ← 各ツール（独立デプロイ可能）
packages/ui/             ← 共通UIコンポーネント
```

## デザインルール

- 配色: Nord テーマ（ダーク/ライト切り替え対応）
- フォント: LINE Seed JP（本文）、Space Grotesk（ブランド）、M PLUS 1 Code（等幅）
- コピーボタン: クリックでクリップボードにコピー、フィードバックアニメーション付き
